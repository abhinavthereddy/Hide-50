# Hide-50: Web-based steganography tool.

This repository present a web-based tool that allows users to discreetly transfer sensitive text messages using Steganograhy. The goal of this project is to demonstrate the use of Steganography as a useful tool, using Python Flask, SQL, HTML, CSS and Python. The underlying Steganography algorithm used in this demo belongs to CÃ©dric Bonhomme, and can be found at https://github.com/cedricbonhomme/Stegano.
